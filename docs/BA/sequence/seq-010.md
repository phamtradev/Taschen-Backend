## Receive Stock (Create Batch)

```mermaid
sequenceDiagram
    autonumber
    actor WS as 👤 Warehouse Staff
    participant ISC as 📦 ImportStockController
    participant ISS as 📦 ImportStockService
    participant ISR as 🗄️ ImportStockRepository
    participant ISDR as 🗄️ ImportStockDetailRepository
    participant BR as 🗄️ BatchRepository
    participant BS as 📦 BatchService
    participant NS as 🔔 NotificationService
    participant DB as 💾 MySQL DB

    WS->>ISC: POST /api/import-stocks/{id}/receive?userId={id}
    ISC->>ISS: receiveStock(importStockId, userId)
    ISS->>ISR: findById(importStockId)
    ISR->>DB: SELECT * FROM import_stocks WHERE id = ? AND received = false
    DB-->>ISR: ImportStock
    ISR-->>ISS: ImportStock
    ISS->>ISR: Mark as received
    ISR->>DB: UPDATE import_stocks SET received = true WHERE id = ?
    ISS->>ISDR: findByImportStockId(importStockId)
    ISDR->>DB: SELECT * FROM import_stock_details WHERE import_stock_id = ?
    DB-->>ISDR: List<ImportStockDetail>
    ISDR-->>ISS: List<ImportStockDetail>
    loop For each ImportStockDetail
        ISS->>BR: Create Batch entity
        Note right of ISS: Batch fields:
        Note right of ISS: - batchCode: auto-generated
        Note right of ISS: - quantity: detail.quantity
        Note right of ISS: - remainingQuantity: detail.quantity
        Note right of ISS: - importPrice: detail.importPrice
        Note right of ISS: - supplier: detail.supplier
        Note right of ISS: - book: detail.book
        Note right of ISS: - variant: detail.variant
        Note right of ISS: - importStockDetail: detail
        ISS->>BR: save(batch)
        BR->>DB: INSERT INTO batches (...)
        DB-->>BR: Batch
    end
    ISS->>BS: syncStockToBook(bookId) for each unique book
    BS->>DB: SELECT SUM(remaining_quantity) FROM batches WHERE book_id = ?
    DB-->>BS: totalStock
    BS->>DB: UPDATE books SET stock_quantity = ? WHERE id = ?
    ISS->>NS: notifyStaff("Stock received")
    NS-->>WS: 🔔 Stock imported!
    ISS-->>ISC: ReceiveStockResponse (with created batches)
    ISC-->>WS: 200 OK + ReceiveStockResponse
```
