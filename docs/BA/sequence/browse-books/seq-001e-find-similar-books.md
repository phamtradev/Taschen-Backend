## 1. Find Similar Books

```mermaid
sequenceDiagram
    autonumber
    actor U as 👤 User
    participant BC as 📚 BookController
    participant BES as 🤖 BookEmbeddingService
    participant BER as 🗄️ BookEmbeddingRepository
    participant BR as 🗄️ BookRepository
    participant DB as 💾 MySQL DB

    U->>BC: GET /api/books/{bookId}/similar?limit=10
    BC->>BES: findSimilarBooks(bookId, limit)
    BES->>BER: findByBookId(bookId)
    BER->>DB: SELECT * FROM book_embeddings WHERE book_id = ?
    DB-->>BER: BookEmbedding (vector)
    alt Embedding exists
        BER-->>BES: BookEmbedding
        BES->>BER: findTopSimilar(bookId, limit)
        BER->>DB: Calculate cosine similarity with all other embeddings
        DB-->>BER: List<BookEmbedding> (sorted by similarity)
        BER-->>BES: List<BookEmbedding>
        loop For each BookEmbedding
            BES->>BR: findById(bookId)
            BR->>DB: SELECT * FROM books WHERE id = ? AND is_active = true
            DB-->>BR: Book
            BR-->>BES: Book
        end
        BES-->>BC: List<BookResponse>
        BC-->>U: 200 OK + List<BookResponse>
    else No embedding found
        BER-->>BES: null
        BES-->>BC: IdInvalidException("Book has no embedding")
        BC-->>U: 400 Bad Request
    end
```
