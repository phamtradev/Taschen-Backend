## 4. API Endpoint

```
GET /api/books
Query Params:
  - page (default: 1)
  - size (default: 10)
  - sortBy (default: id)
  - sortDir (asc/desc)
Auth: Không cần (public)
```

---

## 5. Alternative Flows

### 5.1 Empty Result
- Khi không có book nào:
  - Trả về empty page `[]`
  - HTTP 200

### 5.2 Invalid Pagination
- Khi page < 1: mặc định page = 1
- Khi size > 100: giới hạn size = 100
